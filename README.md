# Lazy Mouse 2 (Archived) 🖱️📱

Lazy Mouse is a versatile remote control tool that transforms your Android device into a wireless mouse, keyboard, and full PC controller. Since the official website (`lzmouse.com`) is currently offline, this repository serves as a public host for the last known stable binaries.

## 🚀 Key Features

- **Wireless Mouse & Keyboard**: Full support for left/right click, scrolling, and complex key combinations.
- **Remote Desktop**: View and control your PC screen directly from your phone.
- **Multimedia Control**: Dedicated remotes for Media Players, Web Browsers, and PowerPoint presentations.
- **PC Audio to Phone**: Stream your PC’s audio directly to your mobile device.

## 🛠️ Requirements & Setup

To use Lazy Mouse, you must have both the PC "Receiver" and the Android app connected to the same Wi-Fi network.

### **PC Requirements**
*   **Microsoft .NET Framework 3.5**: This software requires .NET 3.5 to run. 

### **Installation Steps**
1.  **PC Installation**: Download and run the `Lazy_Mouse_2.0.0.1.exe` on your Windows computer.
2.  **Android Installation**: Install the `Lazy_Mouse_1.0.1.2.apk` on your mobile device.
3.  **Connection**: 
    - Ensure both devices are on the same Wi-Fi.
    - Open the mobile app and select your computer from the list to connect.
